📚 Law Clarity Hub
🚀 About

Law Clarity Hub is a web-based AI application built using Lovable to simplify legal information. It helps users understand complex laws and legal documents through clear explanations and intelligent summarization.

✨ Features
AI-powered legal text simplification
Legal document analysis
Question-based legal assistance
Clean and simple user interface
Multi-language support (Telugu, Hindi, etc.)
Voice-based legal assistant

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

📦 Project Structure
Frontend built with React
Backend for handling requests and AI integration
API-based communication for responses


🎯 Purpose

To make legal knowledge accessible, understandable, and user-friendly for everyone.
